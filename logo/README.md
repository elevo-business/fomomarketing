# FOMO Marketing — Logo-Konzepte

6 differenzierte Logo-Entwürfe für die Marketing-Agentur **FOMO Marketing**.
Gemeinsamer Designgedanke: die symmetrische Buchstabengruppe **O · M · O** als
Wiedererkennungsmerkmal von "F-OMO".

## Konzepte
| # | Name | Idee | Primär / Akzent |
|---|------|------|------------------|
| 01 | ECLIPSE | Zweifarbiger Ring | Marineblau / Champagner-Gold |
| 02 | PULSE | Radar-Signal | Schwarz / Koralle |
| 03 | APERTURE | Auge / Blende | Smaragd / Gold |
| 04 | CREST | Wappen-Monogramm | Oxblood / Gold |
| 05 | MOMENTUM | Aufwärtspfeil | Anthrazit / Aqua-Teal |
| 06 | ORBIT | Umlaufender Punkt | Pflaume / Gold |

## Dateien
- `FOMO_Marketing_Logo_Konzepte.pdf` — Präsentation (8 Seiten, A4 quer)
- `svg/` — editierbare Vektorquellen je Seite
- `generate.py` — Generator (Python + cairosvg)

Neu erzeugen: `python3 logo/generate.py`
